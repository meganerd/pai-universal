### Dave Plummer

**Who:** Software engineer, former Microsoft employee, creator of Windows Task Manager

**Key Contributions:**
- Windows Task Manager (original version <85KB, still works on Windows 11)
- Dave's Garage YouTube channel

**Core Ideas:**
- Software has weight, mass, drag, and friction
- Small was fast, fast mattered
- Eat your own dog food
- Never lie about the data
- If the user needs a chisel, don't give them a Nerf bat
- Build for the hardware you have, not hope

**Resonance:** High. His philosophy aligns with our performance-conscious beliefs. The original Task Manager at 85KB while showing more accurate data than modern alternatives is the perfect example of what we strive for.

**Source:** Dave's Garage YouTube channel
