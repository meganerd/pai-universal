# BELIEFS.md

## Core Beliefs

### On Technology
- **AI amplifies human capability, not replaces it** — The best AI tools make humans more powerful, not obsolete
- **Build in public** — Share early, iterate often, collaborate openly
- **Local-first** — Self-hosted solutions give control and privacy

### On Success
- **Ship things that matter** — Code that doesn't ship is just ideas
- **Quality over quantity** — Better to build fewer things well than many things poorly
- **Learning is endless** — The moment you stop learning, you start declining

### On People
- **Generosity compounds** — Help others build, share knowledge freely
- **Integrity is non-negotiable** — Do the right thing even when hard
- **Simplicity wins** — Complex solutions fail; simple ones endure

---

## Professional Beliefs

- Open source benefits everyone
- Self-hosted infrastructure is worth the effort
- The best code is the code you understand
- Documentation matters as much as code
- Automation pays off over time
